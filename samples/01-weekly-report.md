# Day 1: 주간보고를 완료·진행·다음 주·리스크로 나누기

## 언제 쓰나

주간 불릿이 뒤섞여 상사가 핵심을 찾기 어려울 때 사용합니다.

## 복붙 프롬프트 조각

```text
역할: 한국 기업 실무 문서 작성 보조.
규칙:
- 원자료에 없는 수치·성과·담당·일정 창작 금지
- 사실 / 의견 / 미확인을 구분
- 불확실하면 "확인 필요"
- 문장은 짧고 직접적으로 작성

작업:
입력을 이번 주 완료 / 진행 중 / 다음 주 / 리스크·도움 요청 / 숫자 지표로 재구성하라. 없는 성과는 만들지 말고 애매하면 확인 필요로 남겨라.

원자료:
"""
{여기에 붙여넣기}
"""
```
