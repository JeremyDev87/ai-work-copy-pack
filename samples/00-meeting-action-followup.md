# 무료 샘플: 회의 메모 → 액션아이템 + 팔로업 메일

> 가상 입력으로 사용법을 보여 주는 샘플입니다. 실제 고객 후기나 실적 사례가 아닙니다.

## 1. 회의 메모 정리 프롬프트

```text
역할: 회의록 정리 보조.
아래 메모를 안건별로 재구성한다.
규칙:
- 발언자·수치·담당·기한을 임의로 만들지 말 것
- 결정 / 미결 / 액션을 분리
- 모르는 항목은 "확인 필요"
출력:
1) 회의 목적
2) 안건별 논의
3) 결정사항
4) 미결·보류
5) 액션아이템 표 (담당 | 할 일 | 기한 | 완료 기준)

원 메모:
"""
{여기에 메모 붙여넣기}
"""
```

## 2. 액션아이템 품질 검사

```text
아래 액션아이템을 검사한다.
각 항목마다 담당 명확성, 기한 명확성, 완료 기준 측정 가능성을 Y/N으로 표시한다.
빠진 정보는 추측하지 말고 "확인 필요"로 남긴다.
마지막 열에 개선 문장 1개를 쓴다.

액션아이템:
"""
{여기에 붙여넣기}
"""
```

## 3. 팔로업 메일

```text
회의 후 팔로업 메일 초안을 작성한다.
제목 3안과 본문을 출력한다.
본문에는 결정사항 3개 이내, 액션 표, 다음 확인 시점을 포함한다.
톤은 짧고 중립적으로 유지하고, 없는 사실은 추가하지 않는다.

회의록:
"""
{여기에 붙여넣기}
"""
```
